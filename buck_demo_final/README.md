# Interactive Power Electronics Lab – Buck Converter (v1.0)

**What’s in this build**
- White theme, bigger text.
- **Two plots** (voltage & current) with colorful traces.
- Sliders + number fields update **live** (on `input`).
- **Click** a component to add its **voltage**; **⌘/Ctrl+Click** to add its **current**.
- Ideal **CCM** buck model.

**How to run**
1. Download the ZIP and extract.
2. Open `index.html` in a modern browser. No build step needed.
